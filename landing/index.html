What Vigirix Does

Built for Efficiency

Every design decision optimises for sustainability — less compute, less power, more intelligence.

⚡

Motion Pre-filter

AI only runs when pixels change. Static scenes cost near-zero compute — saving ~70% energy.

🔍

Multi-class Detection

Persons, knives, firearms, ammunition — all detected in a single lightweight YOLOv8n pass.

⏳

Loitering Detection

Tracks dwell time. Flags anyone who lingers in restricted areas beyond a configurable threshold.

📡

Edge Computing

Detection runs locally on any device. No footage upload. No cloud dependency. Minimal bandwidth.

🔔

Instant Alerts

Critical threats trigger email, Telegram, and browser push simultaneously — within seconds.

🏗️

Microservice Ready

Edge and server are independent services. Deploy one camera or a thousand — the architecture scales.
