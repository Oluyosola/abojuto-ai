How It Works

Four Steps. Zero Delay.

From camera to your phone in seconds — no expensive infrastructure required.

Step 01
📷

Camera Watches

A regular USB or IP camera feeds live video into AbojutoAI running on any computer, Raspberry Pi, or laptop.

Step 02
🧠

AI Detects

YOLOv8 analyses every frame in real time. The moment a person is spotted above the confidence threshold, an event fires.

Step 03
📡

Alert Sent

The detection is sent to this central server — instantly reaching every subscribed browser, Telegram bot, and email.

Step 04
🔔

You're Notified

A push notification hits your phone or desktop with a snapshot, timestamp, and person count — wherever you are.

Connect a Device

Add Your Camera to the Network

Any AbojutoAI device can push its detections to this central server. Set up takes under 5 minutes.

1

Clone & install AbojutoAI

Run pip install -r requirements.txt on the field device.

2

Set the central server URL in .env

Point CENTRAL_URL at this server so detections are forwarded automatically.

3

Add your API key

Set INGEST_API_KEY in both the field device and this server's .env.

4

Start the device

Run uvicorn main:app --port 8001 — alerts will appear on this page instantly.

POST /api/ingest — Ingest an alert
# From any AbojutoAI field device: curl -X POST http://this-server/api/ingest \ -H "X-API-Key: your-key" \ -H "Content-Type: application/json" \ -d '{ "device_id": "School-Gate-Cam-1", "location": "Chibok Secondary School", "count": 2, "confidence": 0.91 }'
Tip: Use ngrok or a cloud VPS to make this server publicly reachable so field devices anywhere in Nigeria can connect.
